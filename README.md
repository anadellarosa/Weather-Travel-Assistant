# 🌍 Weather Travel Assistant (Backend)

This project is a backend application that helps users not only check the weather, but also plan trips based on weather conditions.

## 🚀 Features

- 🌤️ Current weather by location  
- 📅 Travel planning (single date & date range)  
- ⚠️ Travel risk analysis (Low / Medium / High)  
- 🎒 Smart packing advice  
- 🗺️ Google Maps location link  
- 💾 SQLite database (CRUD operations)  
- 📤 Data export (CSV & JSON)  
- 🧭 Interactive menu  

---

## 🧠 Concept

This is not just a weather application.

It is designed as a **Weather Travel Assistant**, helping users make decisions based on weather conditions.

---

## 🏗️ Tech Stack

- Python  
- SQLite  
- Open-Meteo API (weather + geocoding)  
- Pandas  

---

## ⚙️ How It Works

1. User selects an option:
   - Current weather  
   - Single travel date  
   - Date range  

2. User enters a location  

3. The system:
   - Converts location into coordinates  
   - Retrieves weather data  
   - Calculates travel risk  
   - Generates advice  
   - Stores the request in the database  

---

## 🔄 CRUD Operations

- **Create** → Save weather requests  
- **Read** → View stored requests  
- **Update** → Modify travel dates  
- **Delete** → Remove records  

---

## 📊 Data Export

Users can export stored data into:

- CSV  
- JSON
